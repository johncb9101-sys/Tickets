Configuración inicial — Base de datos Firebase
Paso 1: Ir a console.firebase.google.com → Crear proyecto (nombre libre, sin Analytics)
Paso 2: En el proyecto → Realtime Database → Crear base de datos → Elegir ubicación → Modo prueba (30 días) → Luego cambiá las reglas a las de abajo
Paso 3: Reglas de seguridad (Database → Rules):
{ "rules": { ".read": true, ".write": true } }
Paso 4: En Project settings ⚙️ → General → desplazate abajo → Tu app → si no hay, clic en </> para agregar una web app → copiá el objeto firebaseConfig
